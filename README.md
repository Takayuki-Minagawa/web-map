# WebMap - シンプル地図アプリケーション

シンプルな地図アプリです。OpenStreetMap と Leaflet を用いて動作し、外部課金サービスやAPIキーは不要です。

- 外部サーバー不要（ローカルの簡易HTTPサーバーのみ使用）
- APIキー不要／課金なし

## 機能

- OpenStreetMap を用いた無料の地図表示
- 住所検索（日本語対応、APIキー不要／Nominatim）
- 主要都市（東京・大阪・京都）への移動ボタン
- 地図クリックでマーカー作成（カスタムアイコン・タイトル表示）
- マーカーの一括削除
- マーカー管理（一覧、編集・削除、インポート/エクスポート）
- 地図画像の保存（PNG スクリーンショット）
- クリック位置とズームレベルの表示
- レスポンシブデザイン（モバイル対応）

## 使い方

1. サーバーを起動します:
```bash
python3 server.py
```

2. ブラウザが自動的に開き、`http://localhost:8000` で地図が表示されます。

3. 手動でアクセスする場合は `http://localhost:8000/` を開きます。

## ファイル構成

- `index.html` - メインHTML
- `map.js` - 地図機能の JavaScript
- `style.css` - スタイルシート
- `server.py` - ローカルHTTPサーバー（Python 標準）
- `README.md` - このドキュメント
- `プログラム仕様.md` - 仕様書

## 必要環境

- Python 3.x（標準ライブラリのみ）
- モダンなWebブラウザ（Chrome, Firefox, Safari, Edge）
- インターネット接続（地図タイル、検索API、CDN読込に必要）
